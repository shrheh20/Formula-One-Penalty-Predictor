Place team badge files in this folder using the normalized team slug.

Current naming convention:
- `mclaren.svg`
- `mercedes.svg`
- `red-bull-racing.svg`
- `ferrari.svg`
- `williams.svg`
- `racing-bulls.svg`
- `aston-martin.svg`
- `haas.svg`
- `audi.svg`
- `alpine.svg`
- `cadillac.svg`

Supported extensions:
- `.svg`
- `.png`
- `.webp`

SVG is recommended for the fastest, sharpest badge rendering.
If a specific file is missing, the app falls back to `default-team.svg`.
