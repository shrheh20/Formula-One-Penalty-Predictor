Place custom driver photos in this folder using the driver's three-letter code in lowercase.

Examples:
- `had.webp`
- `nor.png`
- `ver.jpg`

Supported extensions:
- `.webp`
- `.png`
- `.jpg`
- `.jpeg`
- `.svg`

Recommended format:
- square crop
- `160x160` to `240x240`
- `webp` preferred for smaller payloads

If a specific file is missing, the app falls back to `default-driver.svg`.
